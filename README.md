# quiz-app-springboot
# 🎯 Quiz App - Spring Boot Backend

A RESTful Quiz Application backend built using **Spring Boot**, **JPA**, and **MySQL**.

This backend provides APIs to:
- Add questions
- Fetch questions by category
- Create quizzes
- Submit answers
- Calculate quiz score

---

## 🚀 Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- REST APIs
